# Employee Attrition Prediction using Logistic Regression

## Project Overview
This project uses a Logistic Regression Machine Learning model to predict employee attrition based on HR-related attributes such as salary, work experience, department, satisfaction level, and working hours.

The project demonstrates a complete classification workflow including:
- Data preprocessing
- Data visualization
- Feature engineering
- Model training
- Performance evaluation
- Prediction analysis

---

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature scaling and encoding
- Logistic Regression classification
- Model evaluation using classification metrics
- Employee attrition prediction

---

## Machine Learning Workflow
1. Load HR dataset
2. Clean and preprocess data
3. Encode categorical values
4. Split training and testing datasets
5. Train Logistic Regression model
6. Evaluate model performance
7. Predict employee attrition

---

## Evaluation Metrics
The following metrics are used:
- Accuracy Score
- Precision
- Recall
- F1-Score
- Confusion Matrix
